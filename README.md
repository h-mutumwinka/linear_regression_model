# Summative_Mobile_App_Regression_Analysis

Mission

This project predicts student exam performance based on study habits such as hours studied, sleep hours, attendance percentage, and previous academic scores. 
The goal is to understand which factors influence academic performance and help educators identify students who may need additional support.

Dataset Source:
https://www.kaggle.com/datasets/grandmaster07/student-exam-performance-dataset-analysis?resource=download
